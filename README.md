# Portfolio - Fernando Cortes

Portfolio personal usando Astro y Tailwind CSS.

## Comandos

| Comando           | Acción                                    |
| :---------------- | :---------------------------------------- |
| `npm install`     | Instala dependencias                      |
| `npm run dev`     | Servidor de desarrollo en `localhost:4321`|
| `npm run build`   | Build de producción en `./dist/`          |
| `npm run preview` | Vista previa antes de deploy              |

## Estructura
